Expo mobile app for PulseHive.

To run locally:

1. Install dependencies (from repo root):

```bash
cd mobile-app
npm install
```

2. Start Expo:

```bash
npm start
```

Notes:
- The mobile app expects the API gateway at `http://localhost:4000` during development.
- Install Expo Go on your device or use an emulator.
- To add native navigation libraries, follow the React Navigation docs for Expo.
