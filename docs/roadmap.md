# Roadmap

## Upcoming work

### Core product features
- Implement Capitec Business OAuth integration.
- Build full mobile app screens for order capture, payment logging, and matching.
- Add offline sync and conflict resolution for the mobile app.
- Add push notifications for order status changes.
- Complete workflow for payment proof upload and matching.

### Multi-tenant SaaS features
- Tenant onboarding and tenant management.
- User roles: admin, manager, and staff.
- Subscription plans and billing.
- Tenant-specific branding and settings.

### Web Admin enhancements
- Add login and JWT auth flow.
- Connect dashboard to live backend data.
- Export orders and payments to CSV/PDF.
- Add reporting charts and filters.

### Deployment and testing
- Add CI/CD pipelines for tests and deployments.
- Add integration tests for services and API.
- Harden security, validation, and error handling.
- Prepare App Store / Google Play release.
